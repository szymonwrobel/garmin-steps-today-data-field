# Daily Steps Data Field

A Connect IQ data field for Garmin watches that displays your daily step count at a glance.

## What it does

Shows your current daily steps on the watch face as a data field. Built for Garmin Connect IQ 3.0.0+, with support for most modern Garmin watches (Forerunner, Epix, Fenix, Instinct, MARQ, and more).

## Details

- Pulls data from the device's activity monitor
- Renders once per second
- Currently Polish (`pol`) and English (`eng`) support
- No permissions needed
